Οι stored procedures που ειναι μεσα σε αυτον τον φακελο χρησιμοποιήθηκαν για την υλοποιηση καποιων απο τις λειτουργιες που απαιτηθηκαν από τη βάση.
Στον κώδικα όταν γινεται χρήση καποιας απο τις παραπανω stored procedures χρησιμοποιειται η εντολή CallableStatement.
Ολες οι stored procedures καλουνται κατευθειαν απο τη βαση δεδομενων και ΔΕΝ ειναι αποθηκευμενες μεσα στο προτζεκτ.
